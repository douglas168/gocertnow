# Gemini Image Generation Prompt — Citizen Developer Concept

## Prompt

Create a clean, modern flat illustration showing the concept of a "Citizen Developer" in a corporate environment.

**Main subject (foreground, left 60% of frame):**
A friendly-looking young Taiwanese office worker (female, late 20s, business casual attire — blouse and slacks) sitting at a modern desk. She is smiling confidently while working on a large monitor. The monitor screen clearly shows a visual drag-and-drop app builder interface with:
- A canvas area with colorful UI components (buttons, form fields, cards) being arranged
- A left sidebar with a component palette (icons for text, image, button, list)
- A mouse cursor dragging a component onto the canvas
- No visible code or terminal — purely visual

**Secondary subject (background, right 30% of frame):**
Behind a glass partition, two IT department staff (one male, one female, wearing ID badges) are visible at their desks with server monitoring dashboards. One is giving a subtle thumbs-up gesture toward the main subject, symbolizing IT governance and approval (not opposition).

**Visual metaphors:**
- A small shield icon with a checkmark floats near the IT section, representing "sanctioned/approved tools"
- Faint connecting lines (like a flowchart) float between the main subject's screen and the IT area, suggesting governed collaboration
- A small badge or label near the main subject reads "公民開發者" (Citizen Developer)

**Environment:**
- Modern open-plan office with warm lighting
- Clean desks, potted plants, subtle tech company atmosphere
- Large windows with soft natural light in background

## Style Guidance

- **Art style:** Flat vector illustration, similar to Notion or Linear marketing illustrations. Clean lines, no gradients, minimal shadows.
- **Color palette:** 
  - Primary: Deep blue (#1a365d) and teal (#319795) for tech elements
  - Accent: Coral/orange (#ed8936) for interactive elements on screen
  - Background: Warm off-white (#fafaf9) with light gray (#e2e8f0) for walls/partitions
  - Skin tones: Natural, warm
- **Composition:** 16:9 landscape ratio. Main subject at left-center following rule of thirds. IT background at right third.
- **Mood:** Empowering, approachable, professional but not intimidating. The illustration should convey "anyone can build apps" while subtly showing governance exists.
- **Text to include in the image:**
  - "公民開發者" label near the main subject (Traditional Chinese)
  - "IT 治理" small label near the IT section
- **Avoid:** Photorealism, 3D rendering, dark/moody tones, overly complex scenes, any code visible on screens, stereotypical "hacker" imagery

## Usage Context

This image will be used in an educational study guide for the IPAS AI Application Planner certification exam (Taiwan). The target audience is university students and early-career professionals aged 20-35 in Taiwan. The image should feel relatable and encouraging to non-technical learners.

## Dimensions

- Target: 1200 x 675 px (16:9)
- Must be legible at 600px width (mobile view)
