# Gemini Image Prompt: Tool Comparison Infographic

## Prompt

Create a clean, modern infographic comparing 10 generative AI tools in a grid layout. Dark background (#0f172a navy-black). 16:9 aspect ratio.

Layout: 2 rows of 5 cards, each card is a rounded rectangle with subtle border glow.

Each card contains:
- Tool name in English (large, bold, white)
- Primary domain icon at top (pen icon for text, palette for image, speaker for audio, code brackets for code, flag for Taiwan)
- Domain label in Traditional Chinese below the icon
- Pricing badge: green badge "免費方案" for free tier available, red badge "純付費" for paid-only
- One-line differentiator in Traditional Chinese (small text)

Cards in order:
1. ChatGPT — 文本生成 — 免費方案 — "對話式AI助手，月費制"
2. OpenAI API — 文本生成 — 純付費 — "開發者介面，按token計費"
3. Gemini — 多模態 — 免費方案 — "Google生態，多模態AI"
4. Midjourney — 圖像生成 — 純付費 — "照片級AI繪圖，無免費"
5. GPT Image — 圖像生成 — 免費方案 — "內建於ChatGPT"
6. ElevenLabs — 聲音生成 — 免費方案 — "語音合成品質最高"
7. Suno AI — 聲音生成 — 免費方案 — "AI音樂與歌曲創作"
8. GitHub Copilot — 程式碼 — 免費方案 — "IDE擴充套件，行內補全"
9. Cursor — 程式碼 — 免費方案 — "獨立AI IDE，Vibe Coding"
10. TAIDE — 文本生成 — 免費方案 — "台灣國科會繁中LLM"

Color coding for domain icons: blue for text, purple for image, orange for audio, green for code, red for Taiwan.

Title at top in Traditional Chinese: "生成式AI常見工具比較" with English subtitle "GenAI Tool Comparison Card"

Style: flat design, minimal, tech/education aesthetic. Clean sans-serif font. Subtle gradient on cards. Professional look suitable for a certification study guide.

All Chinese text must be in Traditional Chinese (繁體中文), not Simplified.

## Negative Prompts

- No 3D renders or realistic photos
- No cluttered or busy backgrounds
- No watermarks or stock photo artifacts
- No Simplified Chinese characters
- No cartoon or childish style
- No excessive gradients or neon effects
