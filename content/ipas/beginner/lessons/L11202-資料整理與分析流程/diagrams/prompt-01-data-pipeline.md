# Diagram Prompt: Data Pipeline 4-Stage Flow

**For:** Gemini 2.0 Flash image generation
**Use in:** study-guide.md Section 3-1

## Prompt

Create a clean, modern infographic showing a horizontal data pipeline with 4 connected stages flowing left to right. Style: flat design, dark background (#0F172A), rounded cards with subtle gradients.

Stage 1 (blue card): "資料收集 Data Collection" with a download/inbox icon. Subtitle: "買食材"
Stage 2 (green card): "資料清理 Data Cleaning" with a broom/sparkle icon. Subtitle: "洗切備料". Add a small badge "60-80% 時間"
Stage 3 (orange card): "資料分析 Data Analysis" with a magnifying glass/chart icon. Subtitle: "烹調料理"
Stage 4 (purple card): "資料呈現 Data Presentation" with a bar chart/presentation icon. Subtitle: "擺盤上桌"

Connect stages with prominent arrows. Below each stage, show 2-3 key activities in smaller text:
- Stage 1: 內部資料、外部資料、API
- Stage 2: 缺失值、重複值、異常值、不一致
- Stage 3: 描述性、診斷性、預測性、指示性
- Stage 4: 長條圖、折線圖、圓餅圖

Text in Traditional Chinese + English. Dimensions: 1200x500px. No watermarks.
