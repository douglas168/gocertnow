Create a clean infographic showing Feature Engineering (特徵工程) taxonomy as a tree diagram. Style: flat design, dark background (#0F172A), rounded cards.

Top node (large, gold accent): "特徵工程 Feature Engineering" with subtitle "把原始資料變成 AI 更好用的形式"

Branch 1 (blue): "特徵選擇 Feature Selection" with icon of a filter/funnel. Description: "挑出有用的，丟掉無關的". Example: "從30個欄位選出5個重要的"

Branch 2 (green): "特徵萃取 Feature Extraction" with icon of a squeeze/compress. Description: "壓縮精華，保留關鍵". Example: "3000字報告→300字摘要"

Branch 3 (orange): "特徵轉換 Feature Transformation" with icon of a transform/convert arrow. Description: "換個表達方式". Sub-node highlighted with fire emoji: "獨熱編碼 One-Hot Encoding" showing the conversion: 紅/藍/綠 → [1,0,0] / [0,1,0] / [0,0,1]

Text in Traditional Chinese + English. Dimensions: 1200x700px. No watermarks.
