## Prompt

Create a clean, modern pentagon/star diagram showing the 5V characteristics of Big Data.

**Style:** Dark theme (background #0a0a0f or similar dark navy), with glowing accent colors. Modern, flat design with subtle gradients. The overall aesthetic should feel like a tech startup landing page — sleek and professional.

**Layout:** A regular pentagon shape with each vertex representing one of the 5Vs. The pentagon should be centered with connecting lines between all vertices forming a star/web pattern. Each vertex has a glowing node with an icon and label.

**Five vertices (clockwise from top):**

1. **Top (blue glow #3b82f6):** "Volume 巨量性"
   - Icon: Database/stack icon
   - Subtitle: "大"
   - Example: "健保資料庫 2,300萬人"

2. **Upper-right (green glow #22c55e):** "Velocity 時效性"
   - Icon: Lightning bolt / speed icon
   - Subtitle: "快"
   - Example: "即時股價、IoT 感測"

3. **Lower-right (purple glow #a78bfa):** "Variety 多樣性"
   - Icon: Puzzle pieces / mixed shapes
   - Subtitle: "雜"
   - Example: "文字+圖片+影片+數字"

4. **Lower-left (orange glow #f97316):** "Veracity 真實性"
   - Icon: Shield with checkmark
   - Subtitle: "疑"
   - Example: "GIGO 原則"

5. **Upper-left (yellow glow #eab308):** "Value 價值性"
   - Icon: Diamond / gem
   - Subtitle: "值"
   - Example: "數據→商業洞察"

**Additional elements:**
- A mnemonic banner across the bottom: "口訣：大、快、雜、疑、值"
- The connecting web lines between vertices should glow softly, suggesting interconnection
- Center of the pentagon: "大數據 Big Data" in large text with a subtle data stream animation effect
- Bottom-right corner: small text "LevelCert.com | IPAS AI應用規劃師 初級"

**Text:** All in Traditional Chinese with English in parentheses. Clean sans-serif font.

**Dimensions:** 1200x1200 (square), suitable for embedding in a web-based course lesson.

**Do NOT include:** Any watermarks, stock photo elements, or photorealistic humans.
