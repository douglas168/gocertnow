## Prompt

Create a clean, modern 2x2 matrix diagram explaining the four approaches to defining Artificial Intelligence, based on Russell & Norvig's framework.

**Style:** Dark theme (background #0a0a0f or similar dark navy), with colored quadrant sections. Modern, flat design. Each quadrant should have a distinct pastel-on-dark color.

**Layout:** A 2x2 grid with clear labels on both axes:

- **Y-axis (left side):** Top = "思考（Thinking）", Bottom = "行動（Acting）"
- **X-axis (top):** Left = "像人類（Humanly）", Right = "理性地（Rationally）"

**Four quadrants:**

| | 像人類（Humanly） | 理性地（Rationally） |
|---|---|---|
| **思考（Thinking）** | Quadrant 1: 像人類一樣思考（Thinking Humanly）— 認知建模 Cognitive Modeling — Color: soft blue (#60a5fa) | Quadrant 2: 理性地思考（Thinking Rationally）— 邏輯推理 Laws of Thought — Color: soft purple (#a78bfa) |
| **行動（Acting）** | Quadrant 3: 像人類一樣行動（Acting Humanly）— 圖靈測試 Turing Test — Color: soft green (#4ade80) — Add a small star/highlight to indicate this is the most famous approach | Quadrant 4: 像理性主體行動（Acting Rationally）— 理性代理 Rational Agent — Color: soft orange (#fb923c) — Add a small crown/badge to indicate "Russell & Norvig 推薦" |

**Inside each quadrant, include:**
- The Chinese name (large text)
- The English name (smaller text below)
- A one-line description in Chinese
- A small relevant icon (brain for thinking, robot for acting, human silhouette for humanly, logic symbol for rationally)

**Additional elements:**
- A callout arrow pointing to the "Acting Humanly" quadrant saying: "🔥 圖靈測試（Turing Test）在這裡！"
- A callout arrow pointing to the "Acting Rationally" quadrant saying: "⭐ R&N 主張的方向"
- Bottom-right corner: small text "LevelCert.com | IPAS AI應用規劃師 初級"

**Text:** All in Traditional Chinese with English in parentheses. Clean sans-serif font.

**Dimensions:** 1200x1200 (square), suitable for embedding in a web-based course lesson.

**Do NOT include:** Any watermarks, stock photo elements, or photorealistic humans.
