# 三大檢定方法比較表

```
┌────────────────┬──────────────────────────────┬─────────────────────────────┬───────────────────────────────┐
│                │ t-test（t 檢定）               │ chi-square（卡方檢定）        │ F-test / ANOVA               │
├────────────────┼──────────────────────────────┼─────────────────────────────┼───────────────────────────────┤
│ 比較目標        │ 平均數（means）               │ 類別變數的分佈               │ 變異數（variances）           │
│                │                              │ 或獨立性                     │ / 多組平均數                  │
├────────────────┼──────────────────────────────┼─────────────────────────────┼───────────────────────────────┤
│ 資料類型        │ 連續型數值                    │ 類別型（計數）               │ 連續型數值                    │
│                │ (height, score, salary)      │ (yes/no, A/B/C)              │ (yield, score, weight)        │
├────────────────┼──────────────────────────────┼─────────────────────────────┼───────────────────────────────┤
│ 組數            │ 1組 or 2組                   │ 2組以上（列聯表）             │ 3組以上                       │
├────────────────┼──────────────────────────────┼─────────────────────────────┼───────────────────────────────┤
│ H₀ 形式        │ μ₁ = μ₂（兩組平均相等）        │ 兩變數彼此獨立               │ μ₁ = μ₂ = μ₃（所有組平均相等）│
├────────────────┼──────────────────────────────┼─────────────────────────────┼───────────────────────────────┤
│ 統計量          │ t 統計量                      │ χ² 統計量                   │ F 統計量                      │
├────────────────┼──────────────────────────────┼─────────────────────────────┼───────────────────────────────┤
│ scipy 函式     │ ttest_ind()                  │ chi2_contingency()           │ f_oneway()                    │
│                │ ttest_rel()                  │ chisquare()                  │                               │
│                │ ttest_1samp()                │                             │                               │
├────────────────┼──────────────────────────────┼─────────────────────────────┼───────────────────────────────┤
│ 考試口訣        │ t = 平均                      │ χ² = 類別                   │ F = 多組/變異                 │
├────────────────┼──────────────────────────────┼─────────────────────────────┼───────────────────────────────┤
│ 情境範例        │ 男女薪資差異？               │ 性別 vs 購買行為獨立？        │ 3種行銷方案業績差異？          │
│                │ 課前課後成績差異？            │ 地區 vs 滿意度相關？         │ 4種配方良率差異？              │
└────────────────┴──────────────────────────────┴─────────────────────────────┴───────────────────────────────┘
```

## t-test 三種變體比較

```
┌──────────────────┬──────────────────────────────┬─────────────────────────────────────────────┐
│                  │ 情境                          │ scipy 函式                                   │
├──────────────────┼──────────────────────────────┼─────────────────────────────────────────────┤
│ 單樣本 t-test    │ 一組樣本與已知常數比較          │ ttest_1samp(a, popmean)                     │
│ (1-sample)      │ 例：某班平均身高 vs 全國平均    │                                             │
├──────────────────┼──────────────────────────────┼─────────────────────────────────────────────┤
│ 獨立樣本 t-test  │ 兩個不同群體的平均值比較        │ ttest_ind(a, b)                             │
│ (Independent)   │ 例：男生 vs 女生平均成績        │                                             │
├──────────────────┼──────────────────────────────┼─────────────────────────────────────────────┤
│ 配對樣本 t-test  │ 同一群人的前後測比較            │ ttest_rel(before, after)                    │
│ (Paired/Related)│ 例：訓練前 vs 訓練後成績        │                                             │
└──────────────────┴──────────────────────────────┴─────────────────────────────────────────────┘

關鍵判斷：樣本是否「來自同一組人」？
  → 是（前後測、左右眼）→ 配對樣本 ttest_rel()
  → 否（男 vs 女、A組 vs B組）→ 獨立樣本 ttest_ind()
```
