# Type I / Type II Error Decision Matrix

```
                    ┌─────────────────────────────────────────────────┐
                    │           真實情況（Reality）                     │
                    ├────────────────────┬────────────────────────────┤
                    │  H₀ 為真           │  H₀ 為假                    │
                    │  (差異不存在)       │  (差異存在)                  │
  ┌─────────────────┼────────────────────┼────────────────────────────┤
  │ 拒絕 H₀         │  ❌ 型一錯誤        │  ✅ 正確決策                 │
  │ (Reject H₀)    │  Type I Error      │  True Positive             │
  │                │  機率 = α          │  機率 = 1 - β (Power)       │
  │                │  🗣️「誤判無罪」      │  🗣️「正確找出差異」           │
  ├─────────────────┼────────────────────┼────────────────────────────┤
  │ 無法拒絕 H₀     │  ✅ 正確決策         │  ❌ 型二錯誤                 │
  │ (Fail to       │  True Negative     │  Type II Error             │
  │  Reject H₀)    │  機率 = 1 - α      │  機率 = β                   │
  │                │  🗣️「正確確認無差異」 │  🗣️「誤放差異」              │
  └─────────────────┴────────────────────┴────────────────────────────┘

關鍵記憶：
  α (alpha) = Type I Error  = False Positive = 誤拒真 H₀ = 顯著水準
  β (beta)  = Type II Error = False Negative = 誤留假 H₀
  
  口訣：「α 拒真 / β 留假」
```

## 考試常見混淆點

| 混淆點 | 錯誤認知 | 正確理解 |
|--------|----------|----------|
| α 是什麼 | α 是 p-value | α 是預設的顯著水準門檻（如 0.05） |
| Type I 是哪個 | Type I = β | Type I = α（α先，型一先） |
| 降低 α 的代價 | 更安全、沒有代價 | β 會上升（Type II error 增加） |
| Power 是什麼 | Power = α | Power = 1 - β（正確拒絕假H₀的能力） |
