# 拒絕域與 p-value 視覺化

## 雙尾檢定（Two-tailed test）α = 0.05

```
          拒絕域               不拒絕域（接受域）             拒絕域
          α/2 = 0.025                                     α/2 = 0.025
    ┌─────────────┐                                   ┌─────────────┐
    │▓▓▓▓▓▓▓▓▓▓▓▓│                                   │▓▓▓▓▓▓▓▓▓▓▓▓│
    │▓▓▓▓▓▓▓▓▓▓▓▓│                                   │▓▓▓▓▓▓▓▓▓▓▓▓│
    ───────────────────────────────────────────────────────────────
          -∞         -z* = -1.96        0      z* = +1.96         +∞
                         ↑                          ↑
                   臨界值 (critical value)     臨界值 (critical value)

  若 |t統計量| > 1.96  →  落入拒絕域  →  p < 0.05  →  拒絕 H₀
  若 |t統計量| ≤ 1.96  →  留在接受域  →  p ≥ 0.05  →  無法拒絕 H₀
```

## 單尾檢定（One-tailed test, 右尾）α = 0.05

```
                    不拒絕域                          拒絕域
                                                   α = 0.05
                                               ┌─────────────┐
                                               │▓▓▓▓▓▓▓▓▓▓▓▓│
                                               │▓▓▓▓▓▓▓▓▓▓▓▓│
    ───────────────────────────────────────────────────────────
          -∞                       0          z* = +1.645      +∞
                                                    ↑
                                              臨界值 (critical value)

  若 t統計量 > 1.645  →  落入拒絕域  →  p < 0.05  →  拒絕 H₀
```

## p-value 是什麼（直觀說明）

```
  p-value = 假設 H₀ 為真，觀察到「這麼極端或更極端結果」的機率

  p-value 小  →  在 H₀ 成立的前提下，這個結果幾乎不可能發生
             →  懷疑 H₀ 不成立  →  拒絕 H₀

  p-value 大  →  這個結果很常見  →  沒有理由拒絕 H₀

  決策規則：
  ┌──────────────────────────────────────────────────┐
  │  p < α  →  結果顯著  →  拒絕 H₀  (差異存在)      │
  │  p ≥ α  →  結果不顯著  →  無法拒絕 H₀            │
  └──────────────────────────────────────────────────┘

  ⚠️  p-value ≠ H₀ 為真的機率！
  ⚠️  p-value ≠ α！（p 是計算結果，α 是預設門檻）
```

## Pseudocode 追蹤範例

```python
# 題目情境：兩組學生考試成績是否有顯著差異？（α = 0.05）
from scipy.stats import ttest_ind

group_a = [85, 90, 78, 92, 88]
group_b = [75, 70, 82, 68, 79]

t_stat, p_value = ttest_ind(group_a, group_b)
# 假設輸出：t_stat = 2.34, p_value = 0.047

alpha = 0.05

if p_value < alpha:
    print("拒絕 H₀：兩組成績有顯著差異")  # ← 本題輸出這行
else:
    print("無法拒絕 H₀：兩組成績無顯著差異")

# 追蹤步驟：
# 1. p_value = 0.047
# 2. alpha = 0.05
# 3. 0.047 < 0.05 → True
# 4. 輸出：「拒絕 H₀：兩組成績有顯著差異」
```
