# PMF vs PDF vs CDF 視覺比較

## 三種函數一覽

```
  PMF（離散型）           PDF（連續型）           CDF（累積）
  P(X=x) — 長條          f(x) — 曲線密度         F(x) = P(X≤x)

  機率                    密度                    機率
   ^                       ^                       ^
   │    █                  │     .-.               │              ___
   │    █                  │   .'   '.             │           __/
   │  █ █ █                │  /       \            │        __/
   │  █ █ █ █              │ /         \           │     __/
   │█ █ █ █ █ █            │/           \          │  __/
   └──────────→ x          └─────────────→ x       └─/──────────→ x
   0  1  2  3  4           a              b        a              b

  ✅ 各點機率 > 0          ✅ f(x) ≥ 0             ✅ 從 0 爬到 1
  ✅ Σ P(X=x) = 1         ✅ ∫f(x)dx = 1          ✅ 單調遞增
  ✅ P(X=x) 有意義        ❌ P(X=x) = 0           ✅ F(x) = P(X≤x)
```

## 離散型 CDF vs 連續型 CDF

```
  離散型 CDF（階梯狀）         連續型 CDF（平滑）

  1.0 │              ______    1.0 │                  ___
      │           __|          0.9 │               __/
  0.7 │        __|             0.7 │             _/
      │     __|                0.5 │          __/
  0.4 │  __|                   0.3 │        _/
      │__|                     0.1 │     __/
  0.0 └──────────────→ x       0.0 └___/──────────→ x
       0  1  2  3  4                a              b
```

## 對比表

| 屬性 | PMF | PDF | CDF |
|---|---|---|---|
| 適用型態 | 離散型 | 連續型 | 離散型 + 連續型 |
| 函數定義 | P(X = x) | f(x)（密度） | F(x) = P(X ≤ x) |
| 單點 P(X=x) | **> 0** | **= 0** | 不直接代表單點 |
| 函數值範圍 | [0, 1] | **可以 > 1** | [0, 1] |
| 加總/積分 | Σ = 1 | ∫ = 1 | F(+∞) = 1 |
| 圖形外觀 | 離散長條 | 連續平滑曲線 | 階梯 or 平滑上升 |
| scipy 函式 | `.pmf(k)` | `.pdf(x)` | `.cdf(x)` |

## 考試快速記法

```
問 P(X = x)，X 離散  → PMF  → binom.pmf / poisson.pmf
問 P(X = x)，X 連續  → 答案是 0，改想區間
問 P(X ≤ x)          → CDF  → norm.cdf / binom.cdf / poisson.cdf
問 P(a ≤ X ≤ b)      → PDF 面積 or CDF(b) - CDF(a)
```

> **口訣：M 是點名（Mass → 離散），D 是密度（Density → 連續），C 是累積到左邊。**
