# 中央極限定理（CLT）圖解

## 核心概念

```
不管原始母體長什麼樣，
只要樣本數 n 夠大，
樣本平均數 x̄ 的分佈 → 趨近常態分佈

⚠️ CLT 作用對象：樣本平均數 x̄，非原始觀測值
```

## 圖解：從偏態母體到常態抽樣分佈

```
原始母體（右偏，例如均勻或指數分佈）
  機率
   ^
   │\
   │ \
   │  \
   │   \──────
   └──────────────→ x

抽樣分佈隨 n 增大的演變：

n = 2              n = 10             n = 30
機率                機率               機率
 ^                   ^                  ^
 │  /\               │   /\             │    .-""-.
 │ /  \              │  /  \            │   /      \
 │/    \             │ /    \           │  /        \
 └──────→ x̄         └───────→ x̄        └───────────→ x̄
 仍有偏態            接近對稱           幾乎完美鐘形

                         ↓ n 增大
                    越來越接近常態
```

## 關鍵數學

```
x̄ 的抽樣分佈：

  平均數：E(x̄) = μ         （等於母體平均）
  變異數：Var(x̄) = σ²/n    （隨 n 增大而縮小）
  標準差：SE = σ/√n         （Standard Error，標準誤）

當 n 夠大（通常 n ≥ 30）：
  x̄ ~ Normal(μ, σ²/n)
```

## 直覺示意：多組樣本平均數的分佈

```
每次抽 n=30 筆，重複 1000 次，
記錄每次的 x̄：

  x̄₁ = 24.3
  x̄₂ = 25.1      ← 這些 x̄ 排成直方圖
  x̄₃ = 23.8
  ...
  x̄₁₀₀₀ = 24.7

直方圖：
  頻率
   ^
   │        .-"""-.
   │      .'        `.
   │    .'            `.
   │__.'________________`.__
   └────────────────────────→ x̄ 值
        像常態！

即使原始資料是右偏的
```

## 考試陷阱

| 說法 | 對/錯 |
|---|---|
| CLT 說大樣本下每筆資料都會變常態 | ❌ 錯 |
| CLT 說樣本平均數在大樣本下趨近常態 | ✅ 對 |
| n=5 時，CLT 效果已很完整 | ❌ 錯（通常需 n ≥ 30） |
| 原始母體本來就是常態，CLT 的結論仍成立 | ✅ 對（任何母體都適用） |

## 口訣

> **母體不一定常，平均常常會常**
>
> CLT 講的是**樣本平均數（x̄）**，不是原始觀測值。
