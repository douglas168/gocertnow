# 匿名化三層防護階梯圖 (Anonymization Ladder)

## ASCII 階梯圖

```
              更強的隱私保護
                    ▲
                    │
    ┌───────────────────────────────────────┐
    │  t-相近性（t-Closeness）              │  Layer 3
    │  ✅ 防止：相似性攻擊 / 偏態攻擊        │
    │  原理：敏感屬性的分佈要與整體一致       │
    │  需要：已滿足 ℓ-多樣性                │
    └───────────────────────────────────────┘
                    ▲ 建立在此之上
    ┌───────────────────────────────────────┐
    │  ℓ-多樣性（ℓ-Diversity）             │  Layer 2
    │  ✅ 防止：屬性揭露攻擊                │
    │  原理：每個等價類至少有 ℓ 種不同敏感值  │
    │  需要：已滿足 k-匿名性                │
    └───────────────────────────────────────┘
                    ▲ 建立在此之上
    ┌───────────────────────────────────────┐
    │  k-匿名性（k-Anonymity）             │  Layer 1
    │  ✅ 防止：身份揭露攻擊                │
    │  原理：每筆記錄至少與其他 k-1 筆相同  │
    │  弱點：敏感屬性可能全部相同            │
    └───────────────────────────────────────┘
                    │
              較基礎的保護
```

## 各層弱點示例

```
原始資料表（未匿名化）：
┌──────┬──────┬──────────┬──────────┐
│ 姓名  │ 年齡 │ 郵遞區號  │ 疾病     │
├──────┼──────┼──────────┼──────────┤
│ 小明  │ 30   │ 100      │ 糖尿病   │
│ 小華  │ 31   │ 100      │ 糖尿病   │
│ 小美  │ 29   │ 100      │ 高血壓   │
│ 小偉  │ 32   │ 101      │ 心臟病   │
└──────┴──────┴──────────┴──────────┘

k=2 匿名化後（k-Anonymity 滿足，但 ℓ-Diversity 不足）：
┌──────┬──────┬──────────┬──────────┐
│ 姓名  │ 年齡 │ 郵遞區號  │ 疾病     │
├──────┼──────┼──────────┼──────────┤
│ *    │ 30-31│ 100      │ 糖尿病   │ ← 等價類：2人都是糖尿病！
│ *    │ 30-31│ 100      │ 糖尿病   │    ℓ=1，攻擊者知道郵遞區號
│ *    │ 29-32│ 100-101  │ ?        │    30-31歲 → 一定是糖尿病
│ *    │ 29-32│ 100-101  │ ?        │
└──────┴──────┴──────────┴──────────┘

問題：k-anonymity 滿足（k=2），但敏感屬性同質 → 需要 ℓ-diversity（ℓ≥2）
```

## 對應攻擊類型速查

```
攻擊類型          │ 防禦手段          │ 參數意義
─────────────────┼──────────────────┼──────────────────
身份揭露攻擊      │ k-Anonymity      │ k = 最小相同記錄數
  (Identity      │                  │ k=2 → 每人至少有1個分身
   Disclosure)   │                  │
─────────────────┼──────────────────┼──────────────────
屬性揭露攻擊      │ ℓ-Diversity      │ ℓ = 敏感值多樣性下限
  (Attribute     │                  │ ℓ=3 → 每個等價類至少3種不同疾病
   Disclosure)   │                  │
─────────────────┼──────────────────┼──────────────────
相似性攻擊        │ t-Closeness      │ t = 分佈距離上限
  (Similarity    │                  │ t=0.2 → 等價類敏感分佈
   Attack)       │                  │ 與整體分佈相差 ≤ 20%
```
