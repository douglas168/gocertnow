## 📦 盒鬚圖完整解構

### 🎯 盒鬚圖的五個關鍵數字 (Five-Number Summary)

```
        離群值 ●                                    ← 極端異常點（用圓點標示）

    最大值 ┬ ─────────────────────────────          ← 上鬚 (Upper Whisker)
           │                                         = Q3 + 1.5×IQR 或資料中的最大值
           │
      Q3 ┌─┴─────────────────────────────┐         ← 第三四分位數（75%分位）
         │                                 │
         │   ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓   │         ← 盒子 = 中間 50% 的資料
         │                                 │           (IQR = Q3 - Q1)
    中位數│ ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ │         ← 粗線 = Median (50%分位)
         │                                 │
         │   ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓   │
      Q1 └─┬─────────────────────────────┘         ← 第一四分位數（25%分位）
           │
    最小值 ┴ ─────────────────────────────          ← 下鬚 (Lower Whisker)
                                                      = Q1 - 1.5×IQR 或資料中的最小值
        離群值 ●                                    ← 極端異常點
```

---

## 🔍 盒鬚圖的核心組成

### ① **盒子 (Box)** — 中間 50% 的資料範圍
- **上界 = Q3 (第75百分位數)**：有 75% 的資料比這個值小
- **下界 = Q1 (第25百分位數)**：有 25% 的資料比這個值小
- **盒子高度 = IQR (四分位距)** = Q3 - Q1
  - IQR 代表「中間穩健的一半資料」有多分散

### ② **中位數線 (Median Line)** — 盒子中間的粗線
- 把資料排序後正中間的那個值（第50百分位數）
- **不是平均值！** 盒鬚圖完全看不到平均值在哪

### ③ **鬚 (Whiskers)** — 兩根延伸的線
- **上鬚**：延伸到 `Q3 + 1.5×IQR` 或「在這範圍內的最大值」
- **下鬚**：延伸到 `Q1 - 1.5×IQR` 或「在這範圍內的最小值」
- 鬚的長度代表資料的「極端範圍」，但不包含離群值

### ④ **離群值 (Outliers)** — 單獨的圓點 ●
- 超出鬚的範圍的極端值
- 判定標準：
  - 比 `Q3 + 1.5×IQR` 還大，或
  - 比 `Q1 - 1.5×IQR` 還小

---

## ✅ 盒鬚圖能告訴你什麼？

| 能看到 ✅ | 看不到 ❌ |
|----------|----------|
| **中位數** (中間粗線) | ❌ **平均值** (Mean) |
| **四分位數** (Q1, Q3) | ❌ **標準差** (SD) 或變異數 |
| **IQR** (盒子高度) | ❌ **資料的詳細分布形狀** |
| **離群值** (獨立圓點) | ❌ **樣本數量** (N) |
| **偏態方向** (盒子位置) | ❌ **眾數** (Mode) |
| **資料的分散程度** (整體高度) | |

## 📊 如何判斷偏態 (Skewness)？

### 對稱分布 (Symmetric)
```
    ┬
  ┌─┴─┐
  │ ━ │  ← 中位數在盒子正中間
  └─┬─┘
    ┴
```
- **特徵**：盒子左右對稱，上下鬚長度相近
- **平均數 ≈ 中位數**
- **例子**：理想的常態分布資料

---

### 右偏 (正偏態, Right Skew)
```
    ┬────────  ← 上鬚很長（尾巴拉向右）
  ┌─┴─┐
  │━  │        ← 中位數偏向盒子下方
  └─┬─┘
    ┴           ← 下鬚很短
```
- **特徵**：少數超級富豪把平均值往右拉
- **例子**：薪資、房價（大部分人薪水低，少數CEO薪水超高）
- **平均數 > 中位數**
- **資料大多集中在左邊**（低值區），只有少數極大值在右邊

---

### 左偏 (負偏態, Left Skew)
```
    ┬           ← 上鬚很短
  ┌─┴─┐
  │  ━│        ← 中位數偏向盒子上方
  └─┬─┘
────┴          ← 下鬚很長（尾巴拉向左）
```
- **特徵**：少數極低值把平均值往左拉
- **例子**：考試成績（大部分人高分，少數人超低分）
- **平均數 < 中位數**
- **資料大多集中在右邊**（高值區），只有少數極小值在左邊

## 🎯 什麼時候用盒鬚圖？

### ✅ 適合的場景
1. **快速發現離群值** — 一眼就能看到異常點
2. **比較多組資料的分散情況** — 並排多個盒鬚圖比較不同組別（如5個部門的薪資分布）
3. **資料有極端值** — 盒鬚圖基於中位數和四分位數，不受極端值影響
4. **想看偏態方向** — 判斷資料是左偏還是右偏

### ❌ 不適合的場景
1. **需要看平均值和標準差** — 盒鬚圖完全看不到這兩個指標
2. **需要看詳細分布形狀** — 用直方圖 (Histogram) 或分布圖 (Distribution Plot) 更好
3. **資料量太少** — 少於10個資料點時，盒鬚圖意義不大

## ⚠️ 考試常見陷阱

### 陷阱 1：盒鬚圖能顯示標準差？
- ❌ **錯！** 盒鬚圖只顯示**四分位數 (Q1, Q3) 和中位數**，完全看不到標準差
- ✅ 標準差要看數值統計量或誤差棒圖 (Error Bar)

### 陷阱 2：盒鬚圖能顯示平均值？
- ❌ **錯！** 盒子中間的粗線是**中位數 (Median)**，不是平均值 (Mean)
- ✅ 如果需要同時顯示平均值，有些軟體會在盒鬚圖上加一個 ◆ 符號標示

### 陷阱 3：右偏 = 資料集中在右邊？
- ❌ **錯！** 右偏 (Right Skew) = 尾巴拉向右邊 = **資料大多集中在左邊**，只有少數極大值在右邊拉尾巴
- ✅ **正偏態 (Right Skew)** = 平均數 > 中位數（例：薪資、房價）

### 陷阱 4：盒子越大 = 資料越多？
- ❌ **錯！** 盒子大小 (IQR) 代表**資料的分散程度**，不是資料量
- ✅ 盒子大 = 中間 50% 的資料分布很分散；盒子小 = 資料很集中

### 陷阱 5：離群值一定是錯誤值？
- ❌ **錯！** 離群值 (Outlier) ≠ 錯誤值 (Error)
- ✅ 離群值可能是**有意義的罕見事件**（如億萬富翁的薪資），需要根據業務邏輯判斷是否保留

## 💡 白話總結

> **盒鬚圖就像一個「資料健康檢查報告」**：
> - **盒子** = 體檢正常範圍（中間 50% 的人）
> - **中位數線** = 中位數體重（不是平均！）
> - **鬚** = 偏高偏低但還算正常的範圍
> - **離群值圓點** = 需要特別注意的異常值（太胖或太瘦的人）
> - **盒子位置** = 整體偏胖還是偏瘦（偏態）
>
> 但它**看不到**：平均體重是多少（平均值）、大家的體重差異有多大（標準差）、有多少人受檢（樣本數）。

## 📚 實際案例解讀

### 案例 1：薪資分析（右偏分布）

假設你有一家公司 100 人的月薪資料：
- 90 人的薪資在 30,000 ~ 50,000 元之間
- 5 人的薪資在 60,000 ~ 80,000 元之間
- 5 位主管的薪資在 150,000 ~ 300,000 元之間

盒鬚圖會顯示：
```
        ● 300k (離群值：CEO)
        ● 250k (離群值：CFO)
    ┬─────── 80k (上鬚)
  ┌─┴─┐
  │━  │ 40k  ← 中位數（大部分員工薪資）
  └─┬─┘
    ┴─────── 30k (下鬚)
```

- **中位數** ≈ 40,000 元（大部分人的薪資水平）
- **平均數** ≈ 55,000 元（被高薪主管拉高，不代表一般員工）
- **右偏分布**：大多數人薪資集中在左邊（低值），少數高薪主管在右邊拉尾巴
- **離群值**：CEO 和 CFO 的超高薪資被標記為圓點

**結論**：盒鬚圖清楚顯示「大部分員工薪資約 30k-50k，主管薪資明顯偏高」，比只看平均薪資 55k 更能反映真實情況。

---

### 案例 2：考試成績（左偏分布）

假設一個班級 50 人的考試成績：
- 40 人的成績在 80 ~ 100 分之間
- 8 人的成績在 60 ~ 79 分之間
- 2 人的成績在 30 ~ 40 分之間

盒鬚圖會顯示：
```
    ┬─────── 100 (上鬚)
  ┌─┴─┐
  │  ━│ 88   ← 中位數（大部分人高分）
  └─┬─┘
────┴─────── 60 (下鬚)
        ● 35 (離群值：嚴重不及格)
```

- **中位數** ≈ 88 分（大部分學生表現良好）
- **平均數** ≈ 82 分（被低分學生拉低）
- **左偏分布**：大多數人成績集中在右邊（高分），少數低分學生在左邊拉尾巴
- **離群值**：嚴重不及格的學生

**結論**：盒鬚圖顯示「這是一個簡單的考試，大部分人都考得不錯，只有少數人需要補救教學」。
